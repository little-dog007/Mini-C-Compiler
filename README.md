# Mini-C-Compiler
a simple compilers based Mini C

# ENV
Linux   Vscode     g++


# Mini C


keyword: int void if else while return


symbol: + - * / < <= > >= == !=
        = ; , [ ] { } ( ) /* */


default function: 
  
  
  input(x)--> enter an integer to x from the command line
  
  
  output(x)-->output x to command line

git clone


cd Mini-C-Compiler


make


bash test.sh
